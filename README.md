# 버그캡쳐 (Bug Capture)

현재 기준 안정 버전: **STEP 5-1 저장 기능 검증본**

## 실행

1. 저장소 파일을 내려받습니다.
2. `index.html`을 더블클릭합니다.
3. `테스트 이동`으로 좌표 변경을 확인합니다.
4. `저장 테스트`와 `저장 불러오기`를 확인합니다.

## 현재 검증된 기능

- HTML 직접 실행
- 테스트 좌표 이동
- 브라우저 로컬 저장
- 저장 파일 다운로드 대체 방식
- 저장 불러오기

## 브랜치 운영 원칙

- `main`: 희찬이 직접 테스트한 안정 버전만 보관
- `feature/*`: 다음 기능 개발 및 검증용
- 테스트 승인 전에는 `main`에 병합하지 않음
